HTML text-related tags — examples (no CSS)

This page demonstrates common HTML tags for handling text content. There is no CSS here — only semantic HTML.

Headings

h1: Main heading

h2: Section heading

h3: Subsection heading

h4: Sub-subsection heading

h5: Smaller heading
h6: Smallest heading

Paragraphs, line breaks, and thematic breaks

p: A paragraph of text. Paragraphs are block-level and separated by default spacing.

br: A line break
causes the text to continue on the next line.

hr: A thematic break is shown above and below using the hr tag.

wbr: A suggested line break in long words like Supercalifragilisticexpialidocious allows wrapping opportunities without inserting visible characters.

Inline emphasis and importance

em: emphasized text usually conveys stress emphasis; nested em within em increases emphasis.

strong: strong importance indicates strong importance or seriousness.

i: idiomatic text for alternate voice, technical terms, or thoughts.

b: keyword or lead-in for attention without implying extra importance.

u: non-textual annotation (e.g., proper name in Chinese, misspelling) — not generic underline.

s: no longer accurate text for content that is no longer relevant.

mark: Highlight with mark to denote relevance.

small: small print for side comments or legalese.

Edits

ins/del: We removed this phrase and added this one.

Quotations and citations

q: Inline quotation — as the author said, less is more.

blockquote: A longer quotation can be set apart in a block.
— Source titled Design Basics

Code and computer text

code: Inline console.log("hello") within a sentence.

kbd: User input like Ctrl + C.

samp: Sample output: Operation completed successfully.

var: Variables like x and y.

pre: Preformatted text preserves
line breaks and    spaces.

Subscripts and superscripts

H2O, and Einstein's equation E = mc2.

Vocabulary, terms, and references

dfn: The term algorithm is a set of instructions to solve a problem.

abbr: WHO was founded in 1948.

cite: Citing a work like The Pragmatic Programmer.

time: The event starts at 11:59 PM on New Year's Eve.

data: Product ID SKU‑12345 can be machine-readable.

Bidirectional text

bdi: Isolated name: إيان appears correctly within English text.

bdo: Overridden direction: abc 123.

Notes

This page intentionally includes only semantic HTML tags for text and basic structure. No CSS or inline styles are used. Obsolete presentational tags such as font, center, big, and tt are omitted.

Additional text-related tags (obsolete/non-standard)

The following elements are obsolete or non-standard. They are shown for historical reference only — do not use them in new content. Some may still render in browsers but are non-conforming in modern HTML.

Presentational (obsolete)

big: enlarged text (obsolete)

tt: teletype/monospace (obsolete)

strike: struck-through (obsolete)

font: font element (obsolete; presentational)

center:

centered text (obsolete)

Legacy inline semantics

acronym: ASAP (obsolete; use abbr)

nobr: This text should not break into a new line in legacy behavior. (obsolete)

Non-standard (proprietary)

blink: This text was intended to blink (not supported in modern browsers).

marquee: Scrolling text (non-standard)

Not shown due to breaking document parsing but included in the summary table: xmp, listing, and plaintext (all obsolete and non-conforming; they change how the HTML parser treats the rest of the document).
