13. Grid Shorthand Properties

CSS Grid offers shorthand properties for common patterns.

Common Shorthands:

place-items: center; = align-items + justify-items

place-content: center; = align-content + justify-content

place-self: center; = align-self + justify-self

grid: auto-flow / 1fr 1fr 1fr; = grid-auto-flow / grid-template-columns

gap: 20px; = row-gap + column-gap
